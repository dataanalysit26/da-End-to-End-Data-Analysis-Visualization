📈 E-commerce Behavioral & Sales Analysis (Kaggle Case Study)
✨ Project Overview & Description
This project demonstrates an end-to-end data pipeline—from raw data ingestion to final strategic insights. Utilizing a Kaggle e-commerce dataset, the project involved rigorous data cleaning in Google Sheets, advanced data transformation using SQL, and the creation of an interactive Looker Studio dashboard to monitor business KPIs and user behavior.

Category,Tool/Language,Key Function
Data Cleaning,Google Sheets,"Initial raw data ingestion, handling duplicates, and formatting date/currency fields."
Data Transformation,SQL (BigQuery),"Advanced querying to calculate conversion rates, session-based metrics, and behavioral correlations."
Analytical Summary,Google Sheets,Statistical pivot summaries and trend analysis based on SQL-processed data.
Visualization,Looker Studio,Development of an interactive dashboard for performance monitoring and strategic reporting.
