# 📈 E-commerce Behavioral & Sales Analysis (Kaggle Case Study)

## ✨ Project Overview & Description

This project demonstrates an end-to-end data pipeline—from raw data ingestion to final strategic insights. Utilizing a Kaggle e-commerce dataset, the project involved rigorous data cleaning in Google Sheets, advanced data transformation using SQL, and the creation of an interactive Looker Studio dashboard to monitor business KPIs and user behavior.

## 🎯 Project Objectives

* Perform comprehensive **data cleaning and validation** to ensure analytical accuracy and eliminate inconsistencies within the dataset.
* Identify the **top-selling products** and analyze **monthly sales trends** to understand purchasing patterns.
* Calculate **average order value (AOV)** and perform **customer-level purchasing analysis**.
* Determine the **total spending per customer** to better understand customer value distribution.
* Identify the **top 10 revenue-generating products** based on total sales performance.
* Aggregate and analyze **sales performance by country and transaction date** to reveal geographical and temporal trends.
